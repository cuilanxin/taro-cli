# taro2.x

基于 [taro](https://taro-ui.jd.com/) 开发的 H5 & 微信小程序。

## Getting Started

```shell
$ yarn
$ cnpm install
$ npm install
```

## Start the dev server

```shell
$ yarn start  # dev 模式（增加 --watch 参数） 将会监听文件修改。
$ yarn build  # build 模式（去掉 --watch 参数） 将不会监听文件修改，并会对代码进行压缩打包
```      
## Start the mock server

```shell
$ yarn run mock
```
